# flutter_zitherharp

A shared library used for apps of Zither Harp.

## Features

- entities
- exports
- extensions
- packages
- widgets

## Usage
- Declare the package

```yaml
  flutter_zitherharp:
    git: 
      url: https://github.com/zitherharpcommunity/flutter_zitherharp.git
      ref: main
```

- Import the package

```dart
import 'package:flutter_zitherharp/flutter_zitherharp.dart';
```
